## 주요 기능  
Azure 및 Open AI를 활용해 정보를 수집과 동시에 요약·분류·태그화하고, 사용자가 원하는 정보를 필요 시점에 쉽게 찾아 활용
• 정보 수집 및 자동 업로드

• 수집된 정보를 텍스트로 인식

• 정보 구조화 및 분류에 맞게 데이터 저장

• 정보 검색 및 재활용

• 사용자 제어 및 피드백 학습
